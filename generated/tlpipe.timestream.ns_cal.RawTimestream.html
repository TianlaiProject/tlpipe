tlpipe.timestream.ns_cal.RawTimestream¶

class tlpipe.timestream.ns_cal.RawTimestream(files=None, mode='r', start=0, stop=None, dist_axis=0, use_hints=True, comm=None)[source]¶

Container class for the raw timestream data.

The raw timestream data are raw visibilities (the main data) and other data and meta data saved in HDF5 files which are recorded from the correlator.

:param Same as container.BasicTod.:
